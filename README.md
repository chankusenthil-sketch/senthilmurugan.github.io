# senthilmurugan.github.io

Personal portfolio and professional resume website for **Senthilmurugan Sengottuvelan** — Senior Software Development Engineer.

## Live Site

[https://senthilmurugan.github.io](https://senthilmurugan.github.io)

## Features

- Responsive, mobile-first design
- Smooth scrolling navigation
- Hero, About, Experience, Skills, and Contact sections
- Professional blue/white color scheme
- Pure HTML5, CSS3, and vanilla JavaScript — no build step required

## File Structure

```
.
├── index.html   # Main page with all sections
├── styles.css   # Responsive styling
├── script.js    # Smooth scrolling & interactivity
└── README.md    # This file
```

## Deployment (GitHub Pages)

1. Go to **Settings → Pages** in this repository.
2. Under *Source*, select the **main** branch and `/ (root)` folder.
3. Click **Save** — the site will be live at `https://senthilmurugan.github.io` within a few minutes.
